# Iniciação Científica - Mecatrônica Industrial (SENAI)

## Projeto: Monitoramento de Produtividade em Tempo Real
Este repositório contém a documentação e os arquivos técnicos do projeto de iniciação científica focado em Inteligência Artificial e Ciência de Dados aplicadas à indústria.

### 🎯 Objetivo
Desenvolver um dashboard para visualização de indicadores de performance (KPIs) e identificação de anomalias em processos produtivos utilizando técnicas de Data Science.

### 📂 Estrutura do Repositório
- `/docs`: Documentação do projeto (Artigo, Resumo, PDFs).
- `/src`: Código fonte (Scripts Python ou arquivos de dados).
- `/img`: Prints do Dashboard e esquemas do projeto.
- `/data`: Planilhas e bases de dados utilizadas.

### 🛠️ Tecnologias Previstas
- Python  (Tratamento de dados)
- Power BI ou Dash/Plotly (Visualização)
- Git/GitHub (Versionamento)
